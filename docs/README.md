# FeatherTrailMD Documentation

This directory contains the developer documentation and architecture planning for `FeatherTrailMD` (`ft`).

## 📚 Contents

- **`man.md`**: The Developer Manual. Read this to understand the execution flow from the CLI entry point down to the filesystem orchestration. It's written as a line-by-line guide for new contributors.
- **`ref.md`**: The Quick Reference Guide. A condensed cheat sheet of the project structure, core functions, and shared constants.
- **`ActualPlan.md`**: The Technical Roadmap. A living document tracking the granular implementation steps, decisions, and progress of the multiphase plan.
- **`PLAN.md`**: The Strategic Vision. The original high-level 8-phase feature roadmap and philosophy for the project.
- **`CODE_REVIEW.md`**: A historical record of the 2026-03-06 comprehensive code review and the findings that were subsequently resolved in Phase 1 Polish.

*Note: End-user instructions and usage examples are available in the repository root `README.md`.*
